To setup the API key for the app, follow these steps:

1. Create a file named `apikey.dart` in the `lib` directory.
2. Add the following code to the file:
```dart
const String apikey = 'YOUR_API_KEY_HERE';
const String formkey = 'YOUR_FORMESTER_URL_HERE';
```
3. Replace the value of the `apikey` variable wi    th your API key.
4. Replace the value of the `formkey` variable with your Formester URL or Google Form URL.
5. Save the file and u are good to go.
